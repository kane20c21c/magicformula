## ① 점수 산출 spec — BB × 52주 × 레짐 결합표 (±10)

### 버킷 정의
- **BB %B** = (Close - BB_lower) / (BB_upper - BB_lower)
  - 돌파상(>1.0) / 상단(0.8~1.0) / 중간(0.2~0.8) / 하단(0~0.2) / 돌파하(<0)
- **52주 위치** = (Close - 52w_low) / (52w_high - 52w_low)
  - 고점(≥0.8) / 중간(0.2~0.8) / 저점(≤0.2)

### 점수표 (강세장 / 하락조정장)

| BB \ 52주 | 고점 | 중간 | 저점 |
|---|---|---|---|
| **돌파상** | +5 / -5 | +7 / -2 | +10 / 0 |
| **상단** | 0 / -5 | +2 / -5 | +5 / -10 |
| **중간** | 0 / -2 | +2 / -2 | +10 / +10 |
| **하단** | 0 / +8 | +4 / +5 | +10 / +4 |
| **돌파하** | +8 / +10 | +4 / +10 | +10 / 0 |

- 강세장 = quickregime 강세지속·강세약화 / 하락조정 = 조정·하락
- 강세 컬럼은 down베팅(-) 제거 조정 (강세장 상승 흐름과 싸워 손해 → 0).
  Kane 2026-05-30 수동 조정 (돌파상중간 +7, 돌파하중간 +4 등).
- 하락조정 컬럼은 4분면 vs_bench 그대로.

## ② 핵심 발견

### 빈도 (Kane 직관 검증)
- **BB 상단 돌파 시 82.7%가 52주 신고가권** (신고가 = 추세)
- **BB 하단 돌파 시 69.8%가 52주 레인지 내** (일시적 눌림 = 반등 기회)
- 하단 돌파는 상단 돌파의 1/5 빈도 (2.18% vs 10.96%) — 드뭄

### 4분면 × 레짐 (각 칸 forward 상승률, h=5)
- **중간+저점 = 전천후 최강**: 강세 +19.8, 하락 +19.1 (장기 바닥 안정화)
- **돌파하+중간 = 레짐 반전**: 강세 -3.0, 하락 **+18.2** (BB하단 돌파는 하락장만 반등)
- **돌파상+고점 = 레짐 반전**: 강세 +6.6 (신고가추세), 하락 -5.8 (막판강세)
- **중간+고점 = 강세장 함정**: 강세 -6.4 (n=2726, 신고가 소진)

### 단독 vs 결합
- 52주 단독: -2.4pp / BB 단독: 무신호
- **AND 결합: +13~+19pp** (두 지표가 서로 노이즈 제거)

### 같은 신호도 레짐 따라 정반대
- BB 하단 돌파: 강세장=약한종목 이탈(매수금지) / 하락장=과매도 반등(매수)
- 신고가: 강세장=추세지속(매수) / 하락장=막판강세(회피)
