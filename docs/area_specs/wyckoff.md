## 한 줄 결론

**Wyckoff 영역의 가치는 "국면 상태"나 "심리 지표"가 아니라 hillstorm 의
국면 전환 이벤트 (ACC_COMPLETE 매수 / DIST_CONFIRM 매도) 에 있다.
전환 신호 기준 거래 hit_rate 79.7% (전체) / 89.5% (청산완료).
단 다른 4영역과 달리 연속 점수(±10)가 아니라 이벤트 기반 진입/청산 신호.**

---

## ① 신호 spec — 국면 전환 이벤트 (이벤트 기반)

| 이벤트 | hillstorm 조건 | 액션 |
|---|---|---|
| **ACC_COMPLETE** | 최근 N일 70%+ Accumulation + Hope_Vector 양전환 | 다음날 시가 **매수** |
| **DIST_CONFIRM** | 최근 N일 70%+ Distribution + Hope_Vector 음전환 | 다음날 시가 **매도** |
| (참고) MARKUP_WATCH | Markup 중 Hope MA5 3일↓ | 청산 후보 (미채택) |
| (참고) MARKDOWN_WATCH | Markdown 중 공포 소진 | 매수 준비 (미채택) |
| (참고) PANIC_ZONE | Hope < -0.08 + RelVol > 2.5 | 바닥 (h=5 50% 무신호) |

```
진입: ACC_COMPLETE 신호일 → 다음날 시가 매수
청산: DIST_CONFIRM 신호일 → 다음날 시가 매도
```

## ③ 핵심 발견

### 평가 방식이 결론을 뒤집음 (Kane 통찰)
- **횡단면 hit_rate** (국면일 다음 N일): 표준 -3~-6.5pp → "빼자" 오판
- **시계열 거래 추적** (전환 사이클): hit 79.9% → "살리자"
- Wyckoff 는 한 종목을 기간 두고 추적하는 도구. 횡단면 평가는 부적합.

### 국면 상태 vs 전환 이벤트
- 상태 (Markup 매수/Markdown 매도): hit 55.7%, 중앙값 +0.65%
- 전환 (ACC→DIST): hit 79.7%, 중앙값 +13.4%
- 전환 이벤트가 진입/청산 타이밍을 훨씬 정밀하게 잡음.

### 심리 지표는 거래량/방향 중복 (제외)
- Confidence = (종가-시가)/시가 (방향)
- Energy = 상대거래량 (거래량)
- Hope = Confidence × Energy, Anxiety = 변동폭 × 방향부호
- → 4지표 부호가 "그날 몸통 방향" 하나로 결정 → z-score 점수 동일.
  거래량 영역(Q1/Q2)의 재포장. 독립 정보 아님.

### DIST_CONFIRM 의 역설
- DIST_CONFIRM 다음 5일 상승률 69% (천장 신호인데 상승) — 강세장 거짓 천장.
- 하지만 70일 사이클로 보면 그 사이 충분히 먹고 천장 부근 청산 → 작동.

## ④ 한계 / 미해결

1. **청산 신호 지연** — DIST_CONFIRM 이 안 떠서 끝까지 보유하는 케이스에서
   장기 손실 (미청산: 카카오 -34%, 알테오젠 -16%, NAVER -14%).
   → 손절은 hillstorm(longlivevault) 신호 로직 조정 영역. 본 분석 범위 밖.
2. **표본 적음** — 청산완료 38건. 89.5% 는 생존편향. 전체 79.7% 가 정직.
3. **강세장 효과** — 대부분 종목 상승기라 hit 79.7% 가 시장 덕인지 분리 불가.
   단 청산 손실 거의 없음(-5.7%)은 ACC_COMPLETE 가 바닥 잡는 증거.
4. **연속 점수 아님** — 다른 4영역(±10 점수)과 달리 이벤트(특정일) 신호.
   5영역 결합 시 점수화 방식 별도 설계 필요.
