# 황금률만들기

종목 분석 페이지의 **종합 시그널 점수 가중치를 백테스트로 검증**하고 최적 조합을 도출하는 프로젝트.

## 목적

5영역(추세 / 모멘텀 / 거래량 / 변동성·위치 / 심리·Wyckoff)에 대한 가중치를 직관이 아닌 **실제 매매 성과**로 결정한다. KOSPI 알파(초과수익률)를 기준으로 51개 가중치 조합과 3개 진입 규칙을 비교.

## 디렉토리 구조

## 핵심 결정사항 (요약)

| 항목 | 값 |
|---|---|
| 대상 종목 | 10개 (반도체·자동차·조선·플랫폼·바이오 등 분산) |
| 기간 | 최근 6개월 (필요 시 1년) |
| 초기 자본 | 2억원 (종목당 2천만원) |
| 가중치 조합 | 51개 (Basic 5개 / LL1+HL1+BL3 20개 / LL2+HL2+BL1 30개) |
| 진입 규칙 | R1 임계 돌파 / R2 절대 수준 / R3 부호 전환 — 3개 모두 비교 |
| 청산 | 분할 익절 30%/40%/30% + 조건부 시간 청산 |
| 평가 기준 | KOSPI 알파 (Primary) |

자세한 내용은 [docs/backtest_design_v2.md](docs/backtest_design_v2.md) 참조.
